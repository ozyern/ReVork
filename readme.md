## 📖 About

**ReVork** is an elite custom ROM project by [ozyern](https://github.com/ozyern), delivering flawlessly ported builds of **OxygenOS 16** and **ColorOS 16** for OnePlus flagship devices. 

This repository houses the source code for the official ReVork website. Recently rebuilt from the ground up, the site features a striking **"Obsidian" dark aesthetic**, **Aquamorphic 3.0 fluid physics**, and **Specular Glassmorphism** to provide a premium, flagship-level download experience that rivals native operating systems.

> **Live site:** [ozyern.me](https://ozyern.me)

---

## ✨ Features

- 📱 **Direct ROM Downloads** — Clean, versioned access to OxygenOS 16, ColorOS 16, and legacy OS 15 firmware.
- 🌊 **Aquamorphic 3.0 UI** — Liquid sheen hover states, specular edge highlighting, and buttery-smooth CSS Grid expansion accordions.
- ⚡ **Device-Aware Engine** — Custom cinematic cursor tracking for desktop, swapping to snappy, tactile `:active` press physics for mobile users.
- 🎨 **Tailwind CSS Powered** — Completely rewritten for speed using Tailwind utility classes and Vanilla JS.
- 🗂️ **Zero Clutter** — Streamlined, single-purpose hardware pages. No bloat, just downloads.
